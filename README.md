# FinalProject_RicochetArena

Ricochet Arena is a physics-based shooting game where the player aims and shoots a projectile by clicking and dragging. The objective is to destroy all enemies with a limited number of shots. Players can use wall reflections and special bounce pads to reach targets that are not directly accessible. The levels gradually become more challenging and require more strategy.
